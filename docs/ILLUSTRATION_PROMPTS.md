## Style Guidelines (Moneyfly-inspired)

- **Style**: Minimal linear vector, clean black/onyx outlines
- **Perspective**: Slight 3D / isometric tilt
- **Composition**: Free-floating, dynamic arrangements
- **Mood**: Modern, approachable, Web3 fintech – not corporate, not overly cute
- **Elements**: Floating orbs, UI panels, data streams, thin clean lines

---

## Bot Character Guidelines (AgentHaus)

Include **friendly bot characters** in illustrations for better engagement and brand recognition.

**Bot design:**
- **Shape**: Rounded, modular – head + body (like a small robot or chatbot mascot)
- **Features**: Simple face (dots for eyes, subtle smile or neutral), antenna or sparkle
- **Colors**: Forest (#476520) body/structure, Prosperity Yellow (#FCFF52) accents (chest, eyes, antenna), Lavender (#B490FF) for AI glow
- **Style**: Minimal linear outlines, no gradients on bot – flat color fills
- **Personality**: Helpful, confident, approachable – not toy-like, not creepy
- **Variants**: Slight differences per agent type (Payment bot with coin, Trading with chart, Social with chat bubble)

**Use bots to:** wave, point, hold tools, interact with UI, stand beside dashboards, deploy rockets, verify badges.

## Usage Notes

1. **Bot consistency**: Use the same AgentHaus bot design across all images – forest green body, Prosperity Yellow accents – for strong brand recognition.
2. Append `no text` or `no typography` to avoid baked-in labels.
3. For Midjourney/Flux: add `--style raw` or `--v 6` as needed.
4. For DALL·E: include "vector illustration" for cleaner results.
5. Generate at 2:1 or 16:9 for hero sections; 1:1 for cards/icons.
6. Consider generating one "master" bot reference image first, then use it as a style reference for consistency.
